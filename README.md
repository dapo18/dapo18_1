Вариант 17
Имитация отжига для решения системы нелинейных уравнений
Задача: использовать имитацию отжига для решения системы уравнений:
x² + y² = 25
x + y = 7

1. Пошаговое описание работы алгоритма имитации отжига
Инициализация:
Задаем целевую функцию: f(x,y) = (x² + y² - 25)² + (x + y - 7)²
Генерируем случайное начальное решение (x,y) в диапазоне [-10, 10].
Устанавливаем начальную температуру T и коэффициент охлаждения alpha.

Итерационный процесс:
На каждой итерации генерируем новое решение-кандидат:
x_new = x_current + random.uniform(-step, step)
y_new = y_current + random.uniform(-step, step)

Критерий принятия решения:
Вычисляем разность энергий: ΔE = f(x_new,y_new) - f(x_current,y_current).
Если ΔE < 0 (новое решение лучше) - принимаем его.
Если ΔE ≥ 0 - принимаем с вероятностью exp(-ΔE/T).

Охлаждение:
После каждой итерации понижаем температуру: T = T * alpha.
Процесс продолжается пока температура не достигнет минимального значения.

Завершение:
Возвращаем лучшее найденное решение.

2. Оценка временной сложности
Временная сложность: O(k × n), где:
k - количество итераций
n - количество переменных (в нашем случае n=2)

3. Объяснение временной сложности
Сложность O(k×n) объясняется тем, что на каждой из k итераций:
Вычисляется целевая функция для текущего решения - O(1);
Генерируется и оценивается новое решение-кандидат - O(1);
Выполняется проверка критерия принятия - O(1);
Обновляется температура - O(1);
Поскольку количество переменных n=2 постоянно, фактическая сложность составляет O(k).

5. Краткий ответ
Имитация отжига - это вероятностный алгоритм для нахождения глобального минимума. Он эффективен для решения систем нелинейных уравнений, позволяя избегать локальных минимумов за счет механизма постепенного "охлаждения". Алгоритм особенно полезен когда традиционные методы не справляются с многомодальными функциями.

17. Генетические алгоритмы
Объясните основные операции генетического алгоритма: отбор, кроссовер и мутация.

Отбор - это процесс выбора наиболее приспособленных особей из текущей популяции для создания следующего поколения. Особи с лучшими значениями целевой функции имеют higher вероятность быть выбранными.

Кроссовер (скрещивание) - операция, при которой две родительские особи обмениваются генетическим материалом, создавая потомков. Например, при одноточечном кроссовере части хромосом родителей объединяются в новых комбинациях.

Мутация - случайное изменение генов в хромосоме с небольшой вероятностью. Это вносит разнообразие в популяцию и помогает избежать преждевременной сходимости к локальным оптимумам.
